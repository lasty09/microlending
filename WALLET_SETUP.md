\# Connecting to MicroLend



MicroLend runs on the Sepolia Ethereum testnet. To interact with the platform you need a Web3 wallet. You have two options:



\---



\## Option A — MetaMask (Browser Extension)



1\. Go to metamask.io and install the browser extension

2\. Click Create a new wallet and set a password

3\. Save your 12-word Secret Recovery Phrase somewhere safe

4\. Open microlending-nine.vercel.app

5\. Click Connect Wallet and select MetaMask

6\. In MetaMask click the network dropdown, Show test networks, then select Sepolia



\---



\## Option B — Brave Wallet (Built-in, No Extension Needed)



1\. Download Brave Browser from brave.com

2\. Click the wallet icon in the top-right corner

3\. Click Create a new wallet and set a password

4\. Open microlending-nine.vercel.app

5\. Click Connect Wallet and select Brave Wallet

6\. Switch network to Sepolia testnet inside the wallet



\---



\## Get Sepolia ETH (Required for Gas Fees)



Visit any of these faucets and paste your wallet address:

\- sepoliafaucet.com

\- faucet.quicknode.com/ethereum/sepolia



Funds arrive within seconds.



\---



\## Get Test Tokens (Required to Lend or Borrow)



The platform uses a test lending token. Contact the platform admin with your wallet address to receive test tokens before you can deposit or borrow.



\---



\## Quick Start Flow



Install wallet > Switch to Sepolia > Get Sepolia ETH from faucet

> Contact admin for test tokens > Connect on platform

> Request student verification > Start borrowing
